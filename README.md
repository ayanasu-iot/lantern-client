# lantern-client
The lantern using [Flaschen Taschen Protocol](https://github.com/hzeller/flaschen-taschen)

## Requirement
```
requests==2.22.0
picamera==1.13
```

## Usage 
```
python main.py
```

## Install
```
$ pip install -r requirements.txt
```
